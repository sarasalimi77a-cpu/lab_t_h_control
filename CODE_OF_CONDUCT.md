# Code of Conduct

## Project Purpose
This project focuses on the development of a laboratory temperature and humidity monitoring and control system. The goal is to ensure accurate environmental control, data reliability, and safe operation within laboratory conditions.

All contributions should support these objectives and maintain high academic, technical, and ethical standards.

---

## Our Commitment
We are committed to providing a respectful, professional, and collaborative environment for everyone involved in this project. This repository is intended for educational, research, and technical development purposes.

---

## Expected Behavior
Contributors are expected to:
- Communicate respectfully and professionally at all times
- Provide clear, accurate, and well-documented code
- Follow good programming practices and safety considerations
- Respect different levels of technical knowledge and experience
- Report bugs, risks, or inaccuracies responsibly
- Follow academic integrity and properly credit external sources or libraries

---

## Technical Responsibility & Safety
Because this project relates to laboratory environmental control:
- Code must not intentionally bypass safety limits or control logic
- Changes affecting temperature or humidity thresholds must be clearly documented
- Contributors should avoid unsafe configurations that could damage equipment or compromise experiments
- Any experimental or untested features should be clearly labeled

---

## Unacceptable Behavior
The following behaviors are not acceptable:
- Harassment, discrimination, or offensive language
- Submitting misleading, unsafe, or intentionally harmful code
- Removing safety checks without justification
- Plagiarism or misrepresentation of authorship
- Sharing confidential laboratory data without permission
- Uploading malicious software or unauthorized scripts

---

## Scope
This Code of Conduct applies to:
- All source code, documentation, and configuration files
- GitHub issues, pull requests, and comments
- Any discussion or collaboration related to this project

---

## Enforcement
Violations of this Code of Conduct may result in:
- Request for revision or clarification
- Rejection of contributions
- Removal of content
- Temporary or permanent restriction from contributing

Decisions will be made in the interest of safety, professionalism, and project integrity.

---

## Reporting Issues
If you observe behavior or contributions that violate this Code of Conduct or pose a safety risk, please report them through GitHub issues or directly to the project maintainer.

---

## Attribution
This Code of Conduct is designed specifically for this laboratory temperature and humidity control project and follows general best practices in academic and open-source development.
